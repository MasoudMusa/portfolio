# Portfolio Website — Final Draft (Minimal / Professional / Interactive)

## GLOBAL DESIGN SYSTEM
- Theme: Dark-first (Primary: #0B0B0C, Surface: #121214, Text: #EDEDED, Accent: #4F8CFF)
- Typography:
  - Headings: Inter / SF Pro Display (600–700)
  - Body: Inter (400–500)
- Grid: 12-column, max-width 1200px
- Spacing scale: 4 / 8 / 16 / 24 / 40 / 64
- Radius: 12–20px (cards)
- Shadows: Soft, low opacity
